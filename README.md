# Drako-MiUnlock
A tiny client for Xiaomi's unlocking service. Does not require android-tools!

Setup:
```
python -m venv venv
. venv/bin/activate
pip install -r requirements.txt
```

Run:
```
python drako-miunlock.py
```

I have used <https://github.com/offici5l/MiUnlockTool> for reference, but I don't recommend it.

If you have any questions about the unlocking process, please open an issue.
